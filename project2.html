Cathodic Arc PVD Coating Chamber Automation

Role: Automation Engineer  •  Year: 2024  •  Stack: Allen Bradley PLC, Ignition Edge SCADA

Fully Automated Vacuum chamber specialized for Cathodic Arc Deposition of Nickel-Alloy Corrosion Resistant coating made specifically for Chromalloy. The goal of this project was to construct a Turnkey automation system where a recipe process is loaded from an excel spreadsheet and set all devices to their specified set-points while monitoring real time data, logging all historical data, and handling alarms based on preconfigured conditions. The implementation of this automated system has saved 10 hours of operator overhead every other day for Acree Technologies.

This chamber refactored Allen Bradley MicroLogix 1400 PLC's from an obsolete system and incorporated Ignition Edge Control as SCADA, using a Dell Desktop computer running Windows 11 and a touch screen monitor. This system uses 6 RS-232 connections, 15 Analog Input channels, 7 Analog Output channels, and a multitude of digital IO for control of peripheral devices. The PLC's digital output feeds relays and contactors to supply 3-phase power to various power supplies, 2 heaters, a motor, seven pneumatic controlled devices and two pumps.
